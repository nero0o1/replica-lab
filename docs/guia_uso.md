# Manual do Usuário: Conversor de Formulários

Este manual ensina como usar o programa para transformar seus formulários antigos no novo formato do hospital. Siga os passos abaixo:

## 1. Preparação
1. Abra a **Pasta principal** onde este programa está instalado.
2. Procure pela pasta chamada **Entrada**.
3. Pegue os seus **Arquivos de formulário** (os que você quer converter) e cole dentro dessa pasta **Entrada**.

## 2. Como Transformar
1. Volte para a **Pasta principal**.
2. Procure pelo arquivo chamado **Iniciar_Conversor** (ele tem um ícone de engrenagens ou termina com `.bat`).
3. **Dê um clique duplo** nesse arquivo.
4. Uma janela preta vai aparecer mostrando o progresso. Não feche ela.
5. Quando o processo terminar, a janela avisará que tudo está pronto. Você pode apertar qualquer tecla para fechá-la.

## 3. Onde buscar os arquivos novos
1. Agora, abra a pasta chamada **Saida**.
2. O seu novo arquivo estará lá dentro, pronto para ser usado no sistema do hospital.

---
*Dica: Se o programa encontrar algum erro, ele avisará na janela preta. Se isso acontecer, procure o suporte técnico.*
